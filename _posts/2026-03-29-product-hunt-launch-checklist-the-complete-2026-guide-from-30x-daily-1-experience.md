## Before Launch Day

### 1. Product Readiness
- [ ] Landing page is polished and loads fast
- [ ] Demo video is under 2 minutes
- [ ] Onboarding flow is smooth
- [ ] Free tier or trial is available

### 2. Assets Preparation
- [ ] **Thumbnail**: Eye-catching, 240x240px
- [ ] **Gallery images**: 5-8 screenshots showing key features
- [ ] **Demo GIF**: 15-30 second loop of core functionality
- [ ] **Tagline**: Under 60 characters, benefit-focused

### 3. Launch Strategy
- [ ] Schedule for **12:01 AM PST** (competition resets)
- [ ] Hunter lined up (or self-hunt)
- [ ] **Maker comment drafted** (see template below)
- [ ] Support team ready for 24-hour engagement

## The Maker Comment (Critical!)

Your first comment sets the tone. Here's what works based on real #1 products:

**Structure**:
1. Personal greeting with name
2. The problem you solved
3. 3-4 key features with emojis
4. Promo code for PH community
5. **Engagement question** at the end

**Example from a top launch**:
```plaintext
Hi Product Hunt! 👋

I'm [Name], and I built [Product] because [problem].

Here's what makes it different:
✅ Feature 1
✅ Feature 2
✅ Feature 3

Use code HUNT20 for 20% off.

What feature would you try first? 👇
```
